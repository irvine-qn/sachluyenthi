# sachluyenthi

**sachluyenthi** là một website giới thiệu các đầu sách luyện thi với giao diện trực quan, hiện đại, tập trung vào việc thể hiện kỹ năng **Front-End (HTML & CSS)**.  
Project không sử dụng JavaScript, toàn bộ trải nghiệm được xây dựng bằng layout, styling và tương tác thuần CSS.

## Mục tiêu dự án
- Showcase kỹ năng thiết kế UI/UX Front-End
- Luyện tập tổ chức layout, typography và responsive design
- Xây dựng website tĩnh có cấu trúc rõ ràng, dễ mở rộng
- Trình bày thông tin sản phẩm một cách trực quan và dễ đọc


## Tính năng chính
- Giao diện đẹp, bố cục rõ ràng, dễ sử dụng
- Hiển thị danh sách sách luyện thi theo từng mục
- Mỗi cuốn sách bao gồm:
  - Hình ảnh minh họa
  - Giá bán
  - Ưu điểm và nhược điểm
  - Nút dẫn tới file PDF hoặc link mua sách
- Website hoàn toàn **không sử dụng JavaScript**
- Hoạt động tốt trên trình duyệt, phù hợp làm project portfolio

## Các trang chính
- **Trang chủ**  
  Giới thiệu tổng quan và điều hướng tới các danh mục sách

- **Luyện thi lớp 12**  
  Tổng hợp sách ôn luyện chương trình THPT

- **Luyện thi IELTS**  
  Các đầu sách luyện IELTS kèm đánh giá và giá bán

- **Luyện thi ĐGNL (Đánh giá năng lực)**  
  Sách luyện thi ĐGNL với thông tin chi tiết từng cuốn
